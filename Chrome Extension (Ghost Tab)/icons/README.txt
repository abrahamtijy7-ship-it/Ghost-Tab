Icon Files Needed

Please create the following icon files:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)  
- icon128.png (128x128 pixels)

You can:
1. Use an online icon generator
2. Create simple icons with a snowflake or ghost theme
3. Use any image editor to create PNG files

The icons should represent the "frozen" or "ghost" concept of the extension.

For quick testing, you can temporarily use any PNG files with these names.
